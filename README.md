# Boilerplate React App V2

This is a fully configured React boilerplate meant to quickly bootstrap new app prototypes.

It includes:
## Frontend
- React
- React Router
- React Redux
- React Dates (AirBnB)
- SCSS

## Backend
- Node
- Express

## Bundler and Compiler
- Webpack
- Babel (for ES6/7 Transpiling)

## Testing
- Jest
- Enzyme

## Package Manager
- Yarn

## Database
- Firebase
